# 🎯 Number Guesser

A fun and interactive number guessing game built with vanilla JavaScript, HTML, and CSS. Play directly in your browser with multiple game modes and difficulty levels!

## 🎮 Game Modes

### Normal Mode
Guess a number between **1-100**. Simple, classic guessing with real-time feedback on whether your guess is too high or too low.

### Range Mode
Choose your difficulty level and guess a number in a **secret range**:
- 🟢 **Easy** — 8-18 range (easier, smaller range)
- 🟡 **Medium** — 20-40 range 
- 🔴 **Hard** — 40-60 range (harder, larger range)

## ✨ Features

- 🎨 Beautiful gradient UI with smooth animations
- 📊 Guess history tracking
- 💬 Real-time feedback (higher/lower hints)
- ⏱️ Attempt counter
- 📱 Fully responsive & mobile-friendly
- 🏠 Home button to switch modes anytime
- 🎯 Multiple difficulty levels

## 🚀 How to Play

1. **Open** `game.html` in your browser (just double-click it!)
2. **Choose a mode** — Normal or Range
3. **Pick difficulty** (if Range Mode)
4. **Start guessing** — Use the feedback to narrow down the number
5. **Win** — Find the secret number and try to do it in fewer attempts!
